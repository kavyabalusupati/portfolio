# Portfolio Website

A personal portfolio website to showcase my projects, skills, and background.

## 📌 Project Overview

This project is a responsive portfolio website designed to present my technical skills, projects, and contact information in a clean and user-friendly interface.

## 🛠 Technologies Used

* HTML
* CSS
* JavaScript

## ⚙️ Features

* Responsive design for different screen sizes
* Clean and modern user interface
* Project showcase section
* Simple navigation

## 📂 Project Structure

* index.html
* style.css
* script.js

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in any web browser
